The Task Manager App is an application for managing daily tasks.

To log into the application, please copy
user.txt and tasks.txt to below location
C:\Users\[UserName]\

Once logged in as an Administrator you can:
- register a new user,
- add a new task to the register,
- view all tasks in the registry,
- view administrator tasks,
- generate reports,
- view statistics.
